# Kairos - RPG Habit Tracker 🚀

Kairos is a premium, RPG-themed habit tracking mobile app built with React Native and Expo. It gamifies your daily routine, turning habits into quests and achievements.

## Features ✨

- **RPG Progression**: Level up and earn XP by completing your daily habits.
- **Perfect Day Streaks**: Track your consistency with an overall streak that rewards you for finishing all tasks.
- **Heatmap Analytics**: Visualize your progress with a "LeetCode-style" activity heatmap.
- **Vault & Rewards**: Unlock unique artifacts and gear as you reach new milestones.
- **Cloud Sync**: Real-time synchronization with Supabase to keep your data safe across devices.
- **Stunning UI**: Modern glassmorphism design with ambient animations and vibrant color palettes.

## Tech Stack 🛠️

- **Frontend**: React Native, Expo, TypeScript
- **Backend**: Supabase (Auth, Database, RLS)
- **Analytics**: dayjs
- **Icons & UI**: Lucide, Expo Vector Icons

## Getting Started 🏁

1.  **Clone the repository**:
    ```bash
    git clone [your-repo-link]
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Environment Variables**:
    Create a `.env` file in the root directory and add your Supabase credentials:
    ```env
    EXPO_PUBLIC_SUPABASE_URL=your_supabase_url
    EXPO_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```
4.  **Run the app**:
    ```bash
    npx expo start
    ```
